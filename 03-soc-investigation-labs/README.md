# SOC Investigation Labs

## Overview
This section contains structured write-ups and documentation from defensive cybersecurity
training labs, such as TryHackMe and Blue Team Labs Online (BTLO). The focus is on developing
investigation mindset, interpreting logs and alerts, and applying sound escalation reasoning
consistent with Tier 1 SOC responsibilities.

## Scope
Activities in this folder emphasize:
- Analyzing alerts and log data in controlled lab environments
- Identifying indicators of potential security incidents
- Documenting findings using SOC-style investigative notes

## Contents
- Lab investigation write-ups
- Supporting screenshots from exercises
- SOC-style documentation summarizing observations and conclusions
