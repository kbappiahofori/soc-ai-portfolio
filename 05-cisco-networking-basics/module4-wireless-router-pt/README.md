# Module 4.4.4: Configure a Wireless Router and Clients
**Course**: Cisco Networking Basics - Junior Cybersecurity Analyst Pathway  
**Date**: May 6, 2026  
**Lab Duration**: ~40 minutes

## What I Did
- Connected coaxial and Ethernet cables between modem, router, and 2 PCs
- Configured router GUI: changed default admin password, limited DHCP to 10 users, set SSID "MyHome" with WPA2 Personal
- Connected wireless laptop and verified internet connectivity for all 3 devices

## SOC Relevance
This lab mirrors real-world home network security assessments. I learned that default credentials are a critical threat and must be changed immediately. Limiting DHCP scope reduces the attack surface for rogue devices. Connectivity testing is the same process used during incident response to verify network restoration.

## Security Observation
If this was a real network, my next step would be to disable WPS and check for firmware updates — both are common entry points for attackers.
