# SOC-AI Portfolio

## Overview
This repository is a public-facing portfolio of my hands-on projects in SOC & Cybersecurity.  
It blends technical labs with AI-assisted documentation to demonstrate practical skills in incident reporting, defensive analysis, and networking fundamentals.

## Structure
Projects are organized into clearly defined folders:

- [01-network-traffic-monitoring](01-network-traffic-monitoring)  
  Cisco Packet Tracer labs with VLANs, ACLs, configs, screenshots, and SOC-style incident reports.

- [02-ai-assisted-soc-reporting](02-ai-assisted-soc-reporting)  
  Incident reports enhanced with AI-driven clarity and methodology notes.

- [03-soc-investigation-labs](03-soc-investigation-labs)  
  TryHackMe and BTLO write-ups documenting investigation thinking, log interpretation, and escalation reasoning.

- [04-web-security-observations](04-web-security-observations)  
  Juice Shop and other web security exercises framed from a defensive SOC perspective.

- [templates](templates)  
  Reusable documentation templates for SOC incident reports, lab write-ups, and Packet Tracer simulations.

## Goals
🎯 Build a public-facing portfolio that demonstrates technical skill, clear documentation, and leadership trajectory in InfoSec.  
This repo is designed to scale as I add more labs and reports over time.
