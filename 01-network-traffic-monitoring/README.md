# Network Traffic Monitoring

## Overview
This section is dedicated to documenting simulated network traffic monitoring activities
relevant to Tier 1 SOC operations. The focus is on understanding baseline network behavior,
recognizing anomalous traffic patterns, and supporting incident-style analysis.

## Scope
Work in this folder will focus on:
- Monitoring and analyzing network traffic in simulated environments
- Applying defensive controls such as segmentation and access restrictions
- Documenting observations using SOC-style reporting approaches

## Tools
- Cisco Packet Tracer
- Router and switch configurations
- Supporting screenshots and configuration excerpts

## Contents
- Packet Tracer configuration files
- Lab notes and topology screenshots
- Incident-style documentation of observations and findings

## Status
This section is under active development.  
Individual simulations and documented exercises will be added incrementally.
